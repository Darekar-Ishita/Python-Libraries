# Data Science Practice Repository

A compact learning repository for Python data science libraries, featuring practice notebooks for Matplotlib, NumPy, Pandas, and Seaborn.

## Structure

- `Matplotlib/` - visualization practice notebooks and example CSV data for plotting.
- `NumPy/` - array computing practice notebooks and a sample Python script.
- `Pandas/` - data manipulation practice notebooks and CSV data files for analysis exercises.
- `Seaborn/` - statistical visualization practice notebooks.

## Purpose

This repository is designed for hands-on practice with foundational Python data science tools. It is useful for learning data loading, exploration, transformation, and visualization.
